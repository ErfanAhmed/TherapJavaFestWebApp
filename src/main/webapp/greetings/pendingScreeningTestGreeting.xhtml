The screening test for Therap Java Fest will start on 20-Sep-2012 08:00pm.

Until then, you can familiarise yourself with the rules and regulations of the screening test:
  • The screening test is comprised of multiple choice questions only, each having a single correct answer.
  • The screening test can only be taken within 20-Sep-2012 08:00pm to 21-Sep-2012 08:00pm.
  • The duration of the screening test is 50 minutes
  • You can only take the screening test once.
  • You will get one question at a time. You will get a fixed period of time to answer it. If you fail to answer it within that time, you will be forwarded to the next question. Each question may have different allotted time.
  • You answer a question by selecting an option and clicking the "Submit Answer" button. After you submit an answer, you will be automatically forwarded to the next question.
  • If you click the "Submit Answer" button without selecting and option, you will be forwarded to the next question.
    Beware! If you do so, it will be counted as an incorrect answer and you will not be able to answer that question again.
  • You do not have the option to skip a question and then return again to answer it.
  • You can manually end the test by clicking the "End Test" button.
    Beware! If you click the "End Test" button you will not be able to take the screening test again.
